## The payment flow

A payment has three steps:

1. **Server** creates an intent with your secret API key (`client.paymentIntents.createPay`/`createTx`).
2. **Browser** completes it with the user's Stellar wallet (`new WebClient().pay(intent)`), producing an on-chain `txHash`.
3. **Server** validates that hash (`intent.validate(txHash)`), which updates the status and fires a webhook.

The network is chosen by the API key type: `dv_` keys hit **testnet**, `prod_` keys hit **mainnet**.
