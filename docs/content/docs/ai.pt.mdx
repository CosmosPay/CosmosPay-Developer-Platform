Esta documentação foi feita para ser consumida por LLMs e agentes de programação (Cursor, Claude, ChatGPT…), não
apenas por humanos. Copiar o Markdown de uma página é uma opção — mas há muito mais.

## Arquivos de contexto prontos

Dois endpoints expõem toda a documentação como texto puro que um LLM pode ingerir de uma só vez:

- `/docs/llms.txt` — um índice compacto: cada página com seu título, URL e descrição.
- `/docs/llms-full.txt` — o texto completo de cada página, concatenado.

```bash
# the index
curl https://dev.cosmospay.lat/docs/llms.txt

# everything, in one file (great for a single prompt or a RAG ingest)
curl https://dev.cosmospay.lat/docs/llms-full.txt
```

## Qualquer página individual como Markdown

Cada página tem um menu de ações (canto superior direito da página) com:

- **Copy Markdown** / **View as Markdown** — o código-fonte Markdown bruto daquela página.
- **Open in ChatGPT** / **Open in Claude** — abre o assistente pré-carregado com aquela página.

Você também pode obter o Markdown de uma página diretamente, anexando `/content.md` em `/docs/llms.mdx`:

```bash
# the quickstart page, as Markdown
curl https://dev.cosmospay.lat/docs/llms.mdx/sdk/server/quickstart/content.md
```

<Callout title="Use em um prompt">
Cole o `llms-full.txt` (ou o Markdown de uma única página) no seu assistente e deixe-o escrever a
integração — por exemplo, *"Usando isto, crie um checkout Cosmos Pay que cobra 10 USDC e verifica
o pagamento com um webhook."*
</Callout>

## O SDK traz sua própria documentação para LLMs

O pacote `@cosmosapp/pay_sdk` inclui uma pasta `llms/` legível por IA, então, assim que você o instala,
a documentação está ali em `node_modules` — sem necessidade de rede:

```text
node_modules/@cosmosapp/pay_sdk/llms/
├─ llms.txt          # index
├─ llms-full.txt     # everything in one file
└─ NN-*.md           # one atomic topic per file
```

```bash
# point your agent at the installed SDK docs
cat node_modules/@cosmosapp/pay_sdk/llms/llms-full.txt
```
