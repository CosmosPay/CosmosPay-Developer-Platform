Esta documentación está hecha para ser consumida por LLMs y agentes de programación (Cursor, Claude, ChatGPT…), no
solo por humanos. Copiar el Markdown de una página es una opción — pero hay mucho más.

## Archivos de contexto listos para usar

Dos endpoints exponen toda la documentación como texto plano que un LLM puede ingerir de una sola vez:

- `/docs/llms.txt` — un índice compacto: cada página con su título, URL y descripción.
- `/docs/llms-full.txt` — el texto completo de cada página, concatenado.

```bash
# the index
curl https://dev.cosmospay.lat/docs/llms.txt

# everything, in one file (great for a single prompt or a RAG ingest)
curl https://dev.cosmospay.lat/docs/llms-full.txt
```

## Cualquier página individual como Markdown

Cada página tiene un menú de acciones (arriba a la derecha de la página) con:

- **Copy Markdown** / **View as Markdown** — el código fuente Markdown en crudo de esa página.
- **Open in ChatGPT** / **Open in Claude** — abre el asistente precargado con esa página.

También puedes obtener el Markdown de una página directamente añadiendo `/content.md` bajo `/docs/llms.mdx`:

```bash
# the quickstart page, as Markdown
curl https://dev.cosmospay.lat/docs/llms.mdx/sdk/server/quickstart/content.md
```

<Callout title="Úsalo en un prompt">
Pega `llms-full.txt` (o el Markdown de una sola página) en tu asistente y déjalo escribir la
integración — p. ej. *"Usando esto, crea un checkout de Cosmos Pay que cobre 10 USDC y verifique
el pago con un webhook."*
</Callout>

## El SDK incluye su propia documentación para LLMs

El paquete `@cosmosapp/pay_sdk` incluye una carpeta `llms/` legible por IA, así que una vez que lo instales
la documentación está justo ahí en `node_modules` — sin necesidad de red:

```text
node_modules/@cosmosapp/pay_sdk/llms/
├─ llms.txt          # index
├─ llms-full.txt     # everything in one file
└─ NN-*.md           # one atomic topic per file
```

```bash
# point your agent at the installed SDK docs
cat node_modules/@cosmosapp/pay_sdk/llms/llms-full.txt
```
