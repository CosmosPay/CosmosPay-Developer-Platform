Cette documentation est conçue pour être consommée par des LLM et des agents de codage (Cursor, Claude, ChatGPT…), et pas
seulement par des humains. Copier le Markdown d'une page est une option — mais il y a bien plus.

## Fichiers de contexte prêts à l'emploi

Deux points de terminaison exposent l'ensemble de la documentation en texte brut qu'un LLM peut ingérer d'un seul coup :

- `/docs/llms.txt` — un index compact : chaque page avec son titre, son URL et sa description.
- `/docs/llms-full.txt` — le texte complet de chaque page, concaténé.

```bash
# the index
curl https://dev.cosmospay.lat/docs/llms.txt

# everything, in one file (great for a single prompt or a RAG ingest)
curl https://dev.cosmospay.lat/docs/llms-full.txt
```

## N'importe quelle page seule en Markdown

Chaque page possède un menu d'actions (en haut à droite de la page) avec :

- **Copy Markdown** / **View as Markdown** — la source Markdown brute de cette page.
- **Open in ChatGPT** / **Open in Claude** — ouvre l'assistant préchargé avec cette page.

Vous pouvez aussi récupérer le Markdown d'une page directement en ajoutant `/content.md` sous `/docs/llms.mdx` :

```bash
# the quickstart page, as Markdown
curl https://dev.cosmospay.lat/docs/llms.mdx/sdk/server/quickstart/content.md
```

<Callout title="Utilisez-le dans une invite">
Collez `llms-full.txt` (ou le Markdown d'une seule page) dans votre assistant et laissez-le écrire
l'intégration — par ex. *« À partir de ceci, créez un paiement Cosmos Pay qui facture 10 USDC et vérifie
le paiement avec un webhook. »*
</Callout>

## Le SDK fournit sa propre documentation pour LLM

Le paquet `@cosmosapp/pay_sdk` embarque un dossier `llms/` lisible par l'IA, de sorte qu'une fois installé,
la documentation se trouve directement dans `node_modules` — aucun réseau requis :

```text
node_modules/@cosmosapp/pay_sdk/llms/
├─ llms.txt          # index
├─ llms-full.txt     # everything in one file
└─ NN-*.md           # one atomic topic per file
```

```bash
# point your agent at the installed SDK docs
cat node_modules/@cosmosapp/pay_sdk/llms/llms-full.txt
```
