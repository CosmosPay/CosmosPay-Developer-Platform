A Stellar SEP-7 payment intent. Wraps the raw API payload with typed accessors and atomic action methods so you can operate on it directly: `await intent.validate(txHash)`, `await intent.cancel()`, `await intent.delete()`.

## Constructor

### PaymentIntent

```ts
constructor(client: Client, data: PaymentIntentData)
```

**Parameters**

- `client` — [`Client`](/sdk/definitions/server/client)
- `data` — `PaymentIntentData`

Validate a submitted transaction against this intent. On a match the status is finalized server-side and the webhook fires.

