### process()

Verify a raw incoming webhook request and dispatch it to the listeners registered via &#123;@link WebhookManager.on&#125;. Returns the parsed event. Emits, in order: the raw type (`PAYMENT_INTENT_SUCCEEDED`), the camelCase alias (`paymentIntentSucceeded`) and the catch-all `event`.

```ts
process(rawBody: string | Uint8Array, signatureHeader: string | null | undefined, options?: WebhookProcessOptions): WebhookEvent<PaymentIntentData>
```

**Parameters**

- `rawBody` ‚Äî `string | Uint8Array`
- `signatureHeader` ‚Äî `string | null | undefined`
- `options` _(optional)_ ‚Äî `WebhookProcessOptions`

**Returns** `WebhookEvent<PaymentIntentData>`

### createHandler()

Build a Node.js `http`/`https` request handler that verifies the signature, dispatches the event to your listeners, and replies `200`/`400`.

```ts

```

**Parameters**

- `options` _(optional)_ ‚Äî `WebhookHandlerOptions`

**Returns** `(req: IncomingMessage, res: ServerResponse) => void`

### middleware()

Express-style middleware. Expects the **raw** body (e.g. mounted with `express.raw({ type: '*‚Äã/*' })`). On success it attaches the parsed event to `req.cosmosEvent` and calls `next()`; on a bad signature it replies `400`.

```ts
middleware(options?: WebhookHandlerOptions): (req: RequestLike, res: ResponseLike, next: (err?: unknown) => void) => void
```

**Parameters**

- `options` _(optional)_ ‚Äî `WebhookHandlerOptions`

**Returns** `(req: RequestLike, res: ResponseLike, next: (err?: unknown) => void) => void`
