A Stellar path-payment swap. Wraps the raw API payload with typed accessors and atomic action methods so you can operate on it directly: `await swap.fetch()`, `await swap.submit({ signedXdr })`.

```ts
constructor(client: Client, data: SwapData)
```

**Parameters**

- `data` — `SwapData`

## Methods

### fetch()

Re-fetch this swap from the API, returning a fresh instance.

```ts
fetch(): Promise<Swap>
```

**Returns** `Promise<Swap>`

### submit()

Relay the signed transaction for this swap. Mutates this instance from the refreshed swap in the outcome and returns the outcome.

```ts
submit(options: SubmitSwapOptions): Promise<SwapSubmitOutcome>
```

**Parameters**

- `options` — `SubmitSwapOptions`

**Returns** `Promise<SwapSubmitOutcome>`
