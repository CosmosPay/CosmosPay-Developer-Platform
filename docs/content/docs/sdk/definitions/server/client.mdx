The main entry point of the SDK — a single, atomic client that exposes every resource through its own manager — an atomic, single-client design.

## Constructor

### Client

```ts
constructor(options: ClientOptions)
```

**Parameters**

- `options` — `ClientOptions`

## Properties

### version

Library version.

```ts
static readonly version = "1.1.0"
```

### shared

Process-wide shared defaults maintained by Cosmos Pay (gateway URL, version, internal secrets). Mutate once at startup to change them for every client (e.g. self-hosting): `Client.shared.baseURL = 'https://gateway.your-domain.com'`.

```ts
static readonly shared: SharedConfig
```

### shared

Resolved shared configuration for this client instance.

```ts
readonly shared: SharedConfig
```

### webhookSecret

Default webhook signing secret for the webhook listener (if configured).

```ts
webhookSecret: string | undefined
```

### rest

The underlying HTTP layer. Shared by every manager.

```ts
readonly rest: REST
```

**Type** [`REST`](/sdk/definitions/server/rest)
