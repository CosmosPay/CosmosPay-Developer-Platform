Copy-paste end-to-end patterns. All server snippets assume:

```ts
import { Client, Assets } from '@cosmosapp/pay_sdk';
const client = new Client({ apiKey: process.env.COSMOS_PAY_API_KEY });
```

## 1. Checkout (server creates, browser pays, webhook confirms)

```ts
// server: create the intent for an order
export async function startCheckout(orderId: string, total: string) {
  const intent = await client.paymentIntents.createPay({
    destination: process.env.MERCHANT_ACCOUNT,
    amount: total,
    asset: Assets.USDC,
    reference: orderId,           // ties the intent to your order
    msg: `Order ${orderId}`,
  });
  return intent.toJSON();         // → send to the browser
}
```

```ts
// browser: complete it with the user's wallet
import { WebClient } from '@cosmosapp/pay_sdk/web';
const { txHash } = await new WebClient().pay(intentPayload);
await fetch('/api/confirm', { method: 'POST', body: JSON.stringify({ id: intentPayload.id, txHash }) });
```

```ts
// server: fulfill on the webhook (don't trust the browser alone)
client.webhooks.on('paymentIntentSucceeded', async (event) => {
  await markOrderPaid(event.data.reference, event.data.id);

```

## 3. Idempotent webhook handling

```ts
const processed = new Set<string>(); // use your DB in production
client.webhooks.on('event', (event) => {
  if (processed.has(event.id)) return; // event.id is stable — dedupe on it
  processed.add(event.id);
  handle(event);
});
```

## 4. Reconcile a payment server-side

```ts
// You received a txHash out-of-band (e.g. polled the browser). Finalize it:
const outcome = await client.paymentIntents.validate(intentId, { txHash });
if (!outcome.valid) {
  console.warn('payment not valid:', outcome.reason); // amount/destination/memo mismatch
}
```

## 5. Testnet vs mainnet

```ts
import { Assets, TestnetAssets } from '@cosmosapp/pay_sdk';
// dv_ key → testnet → use TestnetAssets.USDC
// prod_ key → mainnet → use Assets.USDC
const asset = process.env.COSMOS_PAY_API_KEY.startsWith('prod_') ? Assets.USDC : TestnetAssets.USDC;
```
