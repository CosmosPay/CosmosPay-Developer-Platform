O **Cosmos Pay** permite que você aceite pagamentos na rede [Stellar](https://stellar.org) usando
**intenções de pagamento** SEP-7: você descreve um pagamento, o pagador o conclui a partir da própria
carteira e você o confirma on-chain. Em torno desse núcleo você também tem **webhooks**, **produtos**,
**clientes** e **análises**.

Há duas formas de integração, e a maioria dos aplicativos usa ambas em conjunto:

## Como funciona um pagamento

Todo pagamento passa pelas mesmas três etapas:

1. **Seu servidor cria uma intenção** com sua chave de API secreta — por exemplo, `client.paymentIntents.createPay({ ... })`. Você recebe de volta um `uri` SEP-7 e um `qr` para mostrar ao pagador.
2. **O pagador a conclui** na sua carteira Stellar (Freighter, xBull, Rabet, LOBSTR ou Albedo). No navegador isso é uma única chamada: `new WebClient().pay(intent)`, que retorna um `txHash` on-chain.
3. **Seu servidor valida** esse hash — `intent.validate(txHash)` — o que atualiza o status do pagamento e dispara um webhook para seus endpoints.

<Callout title="Testnet vs mainnet">
Você nunca escolhe a rede no código — ela é decidida pelo **prefixo da sua chave de API**: chaves `dv_` rodam
na **testnet**, chaves `prod_` rodam na **mainnet**. Desenvolva com uma chave `dv_` e depois troque por uma
chave `prod_` para entrar em produção.
</Callout>

## Antes de começar

1. **Obtenha uma chave de API** — entre na plataforma para desenvolvedores e gere uma. Veja [Obtenha sua chave de API](/api-keys).
2. **Instale o SDK** — `npm install @cosmosapp/pay_sdk` (Node.js ≥ 18). Veja [Instalação](/sdk/server/installation).
3. **Mantenha a chave no seu servidor.** Ela é secreta; o cliente web no navegador nunca precisa dela.

## Seu primeiro pagamento

```ts
import { Client, Assets } from '@cosmosapp/pay_sdk';

const client = new Client({ apiKey: process.env.COSMOS_PAY_API_KEY });

const intent = await client.paymentIntents.createPay({
  destination: 'G...YOUR_MERCHANT_ACCOUNT',
  amount: '10',
  asset: Assets.USDC,
  msg: 'Order #1001',
});

console.log(intent.uri); // show as a link / button
console.log(intent.qr);  // render this data: URL as an <img>
```
